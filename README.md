# Песочница для HTML/CSS/JS

Базовый учебный проект для практики **HTML, CSS и JavaScript** (1 курс).

## Файлы

- `index.html` — одна страница с 16 независимыми задачами (каждая — `<section>`)
- `styles.css` — минимальные стили (без скруглений)
- `script.js` — заготовка с TODO по каждой задаче

## Как запустить

Открой `index.html` в браузере.

## Заметки для преподавателя

- Каждая задача атомарная и использует `data-js="..."`, чтобы писать JS без переписывания HTML.
- В каждой секции есть подсказка (наведи на `?`) — это стандартный браузерный tooltip (`title`) на русском.
